# ggforestplotR 0.1.0

## Initial release

- Added `ggforestplot()` for coefficient-driven forest plots from tidy data frames or supported model objects.
- Added `add_forest_table()` for one-sided summary-table composition.
- Added `add_split_table()` for traditional split-table forestplot layouts.
- Added grouped sections, striped rows, separator lines, point-shape control, and CI staple-width control.
- Added `as_forest_data()` and `tidy_forest_model()` helpers for preparing plotting-ready coefficient data.
- Added pkgdown documentation, three vignettes, and expanded automated test coverage.
